# 🏏 Cricket Counter

A modern, lightweight, and intuitive cricket scoreboard application for tracking live match statistics in real-time.

Cricket Counter provides essential match scoring features including run tracking, wicket management, over calculation, and player statistics — designed for local matches, practice sessions, and small tournaments.

## 📌 Overview

Cricket Counter simplifies cricket score tracking by providing:

* Real-time scoreboard updates
* Ball-by-ball tracking
* Automated over calculation
* Extras management
* Individual player statistics

The application focuses on performance, usability, and clarity.


## ✨ Features

### 🏏 Match Management

* Start new match
* Reset match
* Configure total overs
* Switch innings support

### 📊 Real-Time Scoring

* Add runs (0–6)
* Add wicket
* Automatic ball count tracking
* Automatic over conversion (6 balls = 1 over)
* Display format: `Overs: 10.3`

### ➕ Extras Handling

* Wide
* No Ball
* Byes
* Leg Byes

### 👤 Player Statistics

* Batsman runs
* Balls faced
* Strike rate (auto-calculated)

### 🎯 Clean UI

* Responsive layout
* Instant score updates
* Simple and accessible controls

## 🎮 Usage

1. Set total overs.
2. Use run buttons (0–6) to update score.
3. Click **Wicket** when a batsman is out.
4. Use extras when required.
5. Monitor automatic over updates.
6. Reset or start a new match when completed.
